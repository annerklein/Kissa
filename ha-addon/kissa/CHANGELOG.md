# Changelog

## [1.0.0] - 2026-01-24

### Added
- Initial release of Kissa Coffee Tracker add-on
- Track coffee beans with origin, process, and tasting notes
- Support for V60, Moka Pot, Espresso, and French Press brewing methods
- Bean-level recipe management with grind settings
- Brew logging and rating system
- World map visualization of coffee origins
- Roaster management with logos
- Persistent SQLite database stored in Home Assistant config
- Full ARM support for Raspberry Pi
